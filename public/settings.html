📧 Email — Gmail Setup
📋 How to get a Gmail App Password:
  1. Go to your Google Account → Security
  2. Enable 2-Step Verification (if not already on)
  3. Search for "App Passwords" → Create one for "Mail"
  4. Copy the 16-character password and paste it above
📱 SMS — Twilio Setup
📋 How to get Twilio credentials (free trial):
  1. Sign up free at twilio.com — get ~$15 trial credit
  2. From the Console Dashboard, copy your Account SID and Auth Token
  3. Go to Phone Numbers → Buy a Number (~$1/month)
  4. Enter the number above in E.164 format: +1XXXXXXXXXX

Cost: ~$0.01 per SMS. 500 texts/month = ~$5.
